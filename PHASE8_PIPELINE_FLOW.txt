                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ METADATA FILTER BUILDING                                │
│ MetadataFilterBuilder                                   │
│ ─────────────────────────────────────────────────────── │
│ • Build ChromaDB-compatible filters                     │
│ • Assessment type, language, category                   │
│ • Job level, industry filtering                         │
└─────────────────────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ STAGE 2: SEMANTIC RETRIEVAL (50-100ms)                  │
│ SemanticSearchService + ChromaDB                        │
│ ─────────────────────────────────────────────────────── │
│ • For each query:                                        │
│   - Convert to embedding                                │
│   - Search ChromaDB with filters                        │
│   - Retrieve top-K chunks (default: 20)                 │
│ • Aggregate results from all queries                    │
│ • Total chunks retrieved: 20-60                         │
└─────────────────────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ STAGE 3: HYBRID RANKING (5-10ms)                        │
│ HybridRanker                                            │
│ ─────────────────────────────────────────────────────── │
│ For each result:                                         │
│   • Semantic Score (weight: 0.5)                        │
│     - From vector similarity                            │
│   • Metadata Score (weight: 0.3)                        │
│     - Assessment type match                             │
│     - Language match                                    │
│     - Job level match                                   │
│     - Industry match                                    │
│   • Skill Score (weight: 0.2)                           │
│     - Overlap: required_skills ∩ assessment_skills      │
│   • Final = 0.5×semantic + 0.3×metadata + 0.2×skills    │
│ • Sort by final ranking score (descending)              │
└─────────────────────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ STAGE 4: DUPLICATE REMOVAL (1-2ms)                      │
│ DuplicateRemover                                        │
│ ─────────────────────────────────────────────────────── │
│ • Group results by assessment_id                        │
│ • Keep highest-ranked chunk per assessment              │
│ • Preserve original ranking order                       │
│ • Result: Unique assessments                            │
└─────────────────────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ STAGE 5: CONTEXT COMPRESSION (2-5ms)                    │
│ ContextCompressor                                       │
│ ─────────────────────────────────────────────────────── │
│ • Limit to top N assessments (default: 5)              │
│ • Format each assessment:                               │
│   - Name, URL, Category, Type                           │
│   - Skills, Competencies, Duration                      │
│   - Job Levels, Languages                               │
│   - Description (truncated to 200 chars)                │
│ • Estimate token count (4 chars ≈ 1 token)             │
│ • Output: LLM-ready compressed context                  │
└─────────────────────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐

│ • Validate similarity scores (min: 0.3)                 │
│ • Check metadata completeness                           │
│ • Validate URLs                                         │
│ • Generate warnings (non-blocking)                      │
